This project allowed me to strengthen my skills in building command-line interface (CLI) applications with Python. One of the key aspects was using argparse to efficiently manage multiple commands, which helped me better understand how to structure a user interface in the terminal.

Additionally, I learned how to handle JSON files as a lightweight alternative to databases, implementing functions to read, write, and update data securely. This made me reflect on the importance of data persistence in applications without relying on more complex systems like SQL or NoSQL databases.

A crucial part of the project was the function that ensures the JSON file exists before performing any operations. This guarantees that the application always has a storage space for tasks without requiring manual intervention:

def ensure_json_file_exists():
    if not os.path.exists(JSON_FILE):
        with open(JSON_FILE, "w") as file:
            json.dump({"tasks": []}, file, indent=4)
        print(f"'{JSON_FILE}' File created automatically.")

Another important aspect was understanding the meaning of if __name__ == "__main__":. Before this project, I wasn’t entirely clear on its purpose in Python. I learned that this conditional ensures that the code inside it only runs when the file is executed directly, and not when it is imported as a module in another script. This is essential for organizing code in a modular and reusable way.

if __name__ == "__main__":
    ensure_json_file_exists()
    main()

Here, ensure_json_file_exists() runs at the start to ensure the JSON file is available, and then main() handles the commands entered in the terminal. I learned that this structure is a best practice in Python, as it allows distinguishing between running a script as a standalone program or using it as a module in other projects.

Overall, this project helped me gain a deeper understanding of key concepts such as dunder (double underscore) methods, the importance of modularity, and how to structure Python applications in a more professional manner. These insights will be very useful in future projects, both in developing CLI tools and more advanced applications.
